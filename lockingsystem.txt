locking system:

lock variable:
0 = no player is locked
1 = player locked on left side
2 = player locked on right side

if 0
  show both buttons as "Lock" - colour green
if 1
  show left button as "Unlock" - colour red
  don't show right button
if 2
  dont show left button
  show right button as "Unlock" - colour red


if 0
  btn1 = lock_btn_lock
  btn2 = lock_btn_lock
if 1
  btn1 = lock_btn_unlock
  btn2 = none
if 2
  btn1 = lock_btn_unlock
  btn2 = none
